隱私權保護政策


睿思創新股份有限公司（包括本公司之關係企業（如適用），以下簡稱本公司）為保障您使用Crossbot服務（以下簡稱本服務）之隱私權，本公司將遵循保護個人隱私及個人資料安全之法令要求，承諾保護及尊重您的個人隱私；本公司隱私權保護政策（以下簡稱本政策）一經公告後即生效，其後變更亦同，適用於本服務之使用者，請詳閱本政策以幫助您了解，我們如何蒐集、處理、利用、保護及國際傳輸您的個人資料。
本公司致力於保護您所提供的資料並保持透明。本政策適用於本公司所有網站及本服務，並說明您對於資料之蒐集、處理、利用、保護、國際傳輸、更新及更正等相關事項所享有之選擇，請詳閱以下內容，且一經您使用本服務後，即表示您已同意本政策：
貳拾、個人資料蒐集、處理、利用及國際傳輸
依照我國個人資料保護法之規定，本公司會透過以下方式蒐集您的個人資料：
1. 蒐集來源：
• 當您使用本服務或透過本服務註冊會員、登錄、購買商品服務、參加本服務所舉辦之實體或線上活動、參加本服務所舉辦之行銷、產品宣傳活動，問卷調查或贈獎活動，或進行滿意度調查、獲取意見回饋、回覆您的提問或資訊查詢、提供及改善服務、使用者經驗調查及分析等客戶服務時，您所提供之個人資料。
• 當您瀏覽本公司網站及自本公司所有網站上下載資訊時，本公司會自動接收並記錄您電腦及行動裝置及瀏覽器上之資料，包括系統相關紀錄、IP位址、使用時間、cookie中的資料、軟體和硬體屬性以及瀏覽及點選網頁紀錄、行動裝置的類型與版本資訊、UDID／裝置識別碼以及行動裝置所提供的地理定位資料來判斷使用者的位置以及使用偏好。該等資料通常無法直接或間接辨識為個人資料，但如該等資料足以直接或間接辨識為個人資料時，本公司對於個人資料之蒐集、處理、利用及國際傳輸，則依本政策之規定告知。
• 為了提供或優化本服務，本公司也會透過商業夥伴（包括但不限於代表本公司、或聽從本公司指示之服務廠商、供應商或承包商，下合稱協力廠商）處間接蒐集您的個人資料，或自經授權之第三方處間接蒐集您的個人資料。
2. 蒐集目的：
蒐集之目的在於進行行銷業務、消費者、客戶管理與服務、網路購物及其他電子商務服務及調查、統計與研究分析、非公務機關依法定義務所進行個人資料之蒐集處理及利用、契約、類似契約或其他法律關係事務、國內外交流業務、採購與供應管理、資（通）訊服務、其他經營合於營業登記項目或組織章程所定之業務、其他諮詢與顧問服務。
3. 蒐集類別：
本服務內蒐集之個人資料，依據法務部頒佈之「個人資料保護法之特定目的及個人資料之類別」，包括C001辨識個人者、C003政府資料中之辨識者、C011個人描述、C035休閒生活及興趣、C036生活格調、C038職業、C102約定或契約及其他詳如相關各項活動項目須知或注意事項或相關契約書之內容，並以本公司與您往來之相關業務或服務及自您或第三人處所實際蒐集之個人資料為準。

4. 處理或利用之期間、地區、對象及方式：
• 除蒐集之特定目的、因執行業務所須之必要保存期間（較法規規定期間更長者）、個別契約所定之保存年限或法律另有規定外，以您要求停止使用您的個人資料或本服務停止提供各該相關之服務之日為止。
• 地區：除另有約定外，在符合我國個人資料保護法國際傳輸規定之前提下，您的個人資料將使用於台灣地區。
• 對象：本公司、本公司之關係企業、業務委外機構及其他商業夥伴（包括但不限於協力廠商）或與本公司因業務需要訂有契約關係或業務往來之公私機構或依法有權之機關。我們在上述之特定使用目的範圍內，可能將您所提供的部分個人資料告知本公司的關係企業、協力廠商，且我們會確保本公司的關係企業、協力廠商會嚴格遵守本政策。
• 方式：我們不會在與蒐集目的無關的範圍內處理或利用或國際傳輸您的個人資料予第三人，您的個人資料除用於會員管理、客戶管理之檢索查詢等功能外，亦將利用於金流服務、行銷廣宣等。
5. 您得自由選擇是否提供個人資料予本公司，惟一旦您拒絕提供或所提供之資料不完整，將可能導致無法完整享受本服務或完全無法使用Crossbot服務，例如未提供身分證字號，將無法參與本服務舉辦之行銷等相關活動。本公司提供之產品或服務可能包含其他網站或網頁的連結或使用第三人提供之服務，惟本公司對於此等不屬於本公司之連結或服務不負任何責任，您應了解其蒐集個人資料的行為，包括但不限於其內容或隱私權政策，若因此其他網站或網頁的連結或使用第三人提供之服務致生損害，概與本公司無涉。
6. 本公司無法根據您所提供之資料，確認您的身分真實性或有其他個人資料冒用、盜用、資料不實等情形。如您所提供之資料有任何偽冒不實，應自負一切民、刑事法律責任。
7. 本公司均會依照我國法律規定，直接或間接取得您的個人資料，但如不幸發現您的個人資料遭盜用或冒用於本服務時，請立即依本政策通知本公司，本公司將依照您的請求辦理。

• 其他基於您的同意，於需要時向您發送與我們的業務有關的行銷等其他訊息。
2. 共用資料之方式：

貳貳、Cookie
1. 「Cookie」是指網站伺服器用來和網站瀏覽者使用之瀏覽器進行溝通的一種技術，它可能在您的電腦中隨機儲存字串，用以辨識區別您與其他使用者，Cookie會在您瀏覽網站或頁面時由該網站傳送到您的電腦中，當您再次瀏覽，就能透過 Cookie 辨識您的瀏覽器。Cookie可能會儲存使用者偏好設定和其他資訊。
2. 為提升服務品質，MAAC會在您的電腦或行動裝置上設定並存取MAAC之Cookie資料。MAAC會使用網路beacon存取本公司所有網站與產品及服務有關的Cookie。您可以將瀏覽器重設為拒絕所有Cookie，或是在傳送Cookie時顯示提醒。不過，若不啟用Cookie，有些網站功能或服務可能無法正常運作。

1. 當您使用本服務時，我們會運用這些帳戶資訊驗證您的身分，防止他人未經授權存取您的帳戶。
2. 本公司僅將您的個人資料提供給有「知悉必要」之本公司員工和服務提供者、關係企業或協力廠商，並確保其受嚴格保密要求約束。

4. 本公司已確實採取符合業界標準之實體、電子及程序防護措施。
5. 網際網路及其任何通訊都不能保證絕對安全，本公司無法確定或保證您透過網際網路技術向本公司或第三人提供的個人資料的安全。對此，本公司制定有處理資料安全性疑慮之程序，並會依據應適用之法律要求通知您和任何主管機關。

本公司將保留隨時修訂本政策之權利。如本條款有任何更動時，將於本公司網站上張貼公告，除另有規定外，一經發布便立即生效。

貳伍、個人資料相關權利
1. 依照個人資料保護法第三條規定，您得與本公司客服連絡進行申請，就您的個人資料行使下列權利：

2. 除經應適用之法律禁止，或本公司基於合法目的保留您的個人資料之情形外，本公司會盡合理努力回應前述請求；如有無法回應請求之情形，本公司會及時通知您。於受理您所提出之關於您的個人（資料之任何請求前，本公司有權先確認您的身份。如對於您的個人資料有任何問題，請利用本隱私政策提供之聯繫方式（詳下述）與我們聯繫。
